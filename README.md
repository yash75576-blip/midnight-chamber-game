Midnight Chamber

Project Overview
Midnight Chamber is a turn-based probability strategy game developed using Python and the Pygame library. The game simulates a high-risk decision-making scenario where a player competes against an AI opponent using a revolver loaded with a random mix of live and blank shells.

Objective
The objective of the game is to strategically decide whether to shoot the opponent or take the risk of shooting oneself, while managing limited health points and uncertain probability outcomes.

Features
Turn-based gameplay
AI decision-making based on probability
Timed result reveal system
Enemy thinking delay simulation
Health bar visualization
Extra-turn mechanic for blank self-shots
8-shell randomized chamber (minimum 3 live shells)

Technologies Used
Python 3
Pygame library
State-based game loop architecture
